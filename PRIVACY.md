# Privacy Policy — Clair

**Last updated: April 2026**

Clair is a free, open-source Chrome extension that enhances the Claude AI web interface (claude.ai).

## Data Collection

Clair does **not** collect, store, or transmit any personal data to external servers.

## Data Storage

All data generated by Clair is stored **locally in your browser** using Chrome's `chrome.storage.local` API. This includes:

- User preferences (chat width, sidebar state, keyboard shortcuts, subscription plan)
- Usage statistics (message counts, token estimates for the 5-hour rolling window)
- Conversation bookmarks

This data never leaves your device.

## Permissions

- **storage**: Stores preferences and usage data locally in your browser.
- **downloads**: Enables exporting conversations as files to your local device.
- **activeTab**: Allows the extension to interact with the active claude.ai tab when triggered by the user.
- **scripting**: Injects the Clair UI into claude.ai pages.
- **Host permission (claude.ai)**: The extension only operates on claude.ai and does not access any other websites.

## No Third Parties

Clair does not share any data with third parties. There are no analytics, no tracking, and no advertising.

## Remote Code

Clair does not execute any remote code. All code is bundled within the extension package.

## Open Source

Clair is fully open source. You can review all code at: https://github.com/iljai0725/clair
